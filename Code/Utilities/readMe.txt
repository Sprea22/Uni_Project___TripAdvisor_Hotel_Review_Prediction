script that has been used to merge all the CSV files into a unique onee. 

######################
# Data_Selection.py  #
######################
Script that has been used to select a subset of data from the dataset of all reviews. In particular, the script select 9000 reviews for each "Overall" (1,2,3,4,5), and then it merges all of them together. The result is:
- Training dataset: 31500 reviews
- Testing dataset: 13500 reviews
- Total dataset: 45000 reviews
